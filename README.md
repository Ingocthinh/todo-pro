Ứng dụng quản lý công việc (Todo List) đầy đủ tính năng với React + Vite (Frontend) và Express.js (Backend).

==================================================

## TÍNH NĂNG

### Cơ bản
- Thêm, xóa, chỉnh sửa công việc
- Đánh dấu hoàn thành / chưa hoàn thành
- Tìm kiếm công việc theo từ khóa
- Lọc theo trạng thái (Tất cả / Hoàn thành / Chưa xong)

### Nâng cao
- Ưu tiên công việc: Cao / Trung bình / Thấp
- Date Picker dạng lịch bảng:
  - Hiển thị theo tháng / năm
  - Đánh dấu ngày hiện tại
  - Đánh dấu ngày đã chọn
  - Vô hiệu hóa ngày quá khứ
  - Nút chọn nhanh: Hôm nay, Ngày mai
  - Điều hướng tháng
- Phân loại công việc theo danh mục
- Dashboard thống kê
- Chỉnh sửa toàn bộ thông tin công việc
- Xóa hàng loạt công việc đã hoàn thành
- Cảnh báo công việc quá hạn

==================================================

## BACKEND
- Lưu trữ dữ liệu bằng file JSON

- API thống kê riêng
